# Prior-Art Analysis Skill

## Goal

For each candidate patent direction, compare it against external search results and produce a structured difference analysis.

## Required Columns

- 公开号
- 申请号
- 申请日
- 发明名称
- 申请人
- 摘要
- 差异点

## Difference Analysis Requirements

The difference point should compare:

- application scenario;
- technical object;
- input data or indicators;
- algorithm, control, or evaluation mechanism;
- system modules or deployment flow;
- likely claim focus;
- unresolved verification points.

Do not claim novelty as fact. State that the analysis is preliminary when based only on web snippets.

## Formal Verification Reminder

Web search is only a quick screening step. Before submission, the user must verify patent full text, claims, legal status, and family patents through a formal patent database.
